# 中国可回收火箭与登月工程研究

> 日期：2026-02-11
> 来源：央视新闻, 新浪新闻, Wikipedia

---

## 重大进展：长征十号 + 梦舟飞船首次试验（2026-02-11）

### 试验概况

**时间**：北京时间 2026年2月11日 11:00

**地点**：文昌航天发射场

**试验内容**：
- 长征十号运载火箭系统低空演示验证
- 梦舟载人飞船系统最大动压逃逸飞行试验

**结果**：成功实施海上溅落回收

### 历史性突破

此次试验创造了多项"首次"：

| 首次项目 | 意义 |
|----------|------|
| 长征十号首次初样状态点火飞行 | 验证火箭原型性能 |
| 我国首次飞船最大动压逃逸试验 | 验证紧急逃生系统 |
| 首次载人飞船返回舱海上溅落 | 验证海上回收技术 |
| 首次火箭一级箭体海上溅落 | 验证可回收技术 |
| 文昌发射场新建发射工位首次点火 | 验证新设施能力 |

## 核心装备：长征十号运载火箭

### 火箭概况

| 参数 | 数值 |
|------|------|
| 全称 | Long March 10 (CZ-10) |
| 类型 | 重型运载火箭 |
| 用途 | 载人月球探测 |
| 状态 | 初样阶段（2026年首次飞行） |

### 技术特点

1. **可重复使用设计**
   - 芯一级可回收
   - 符合可重复使用要求

2. **发动机**
   - YF-100K 升级版发动机
   - 使用液氧/煤油推进剂

3. **运载能力**
   - LEO（近地轨道）：14,000kg+（Medium-lift）
   - 专为月球任务设计

### 发展历程

| 时间 | 里程碑 |
|------|--------|
| 早期 | 系留点火试验成功 |
| 2026年前 | 零高度逃逸飞行试验 |
| **2026-02-11** | **首次初样状态点火飞行** |
