## 一、Tesla Optimus (Optimus)

### 1.1 项目概述

**全称**：Optimus（又名 Tesla Bot）
**开发商**：Tesla, Inc.
**发布时间**：2021年8月19日 AI Day
**创始人**：Elon Musk
**定位**：通用人形机器人

**核心愿景**：
- Musk 表示："Optimus 最终可能比 Tesla 的汽车业务更重要"
- 目标：执行"人类不想做的任何事情"
- 预计售价：约 $30,000

### 1.2 发展历程

| 时间 | 里程碑 |
|------|----------|
| **2021.08** | AI Day 首次公布 |
| **2022.04** | Cyber Rodeo 展会展示 |
| **2022.09** | AI Day 2：原型行走和手臂移动演示 |
| **2023.09** | Gen 2：增加灵活性，可做瑜伽姿势 |
| **2023.12** | Gen 2：跳舞、煎蛋 |
| **2024.05** | 工厂任务演示 |
| **2024.10** | We, Robot 活动：互动演示 |
| **2025.03** | 宣布 2026 年送 Optimus 去火星 |
| **2025.06** | 项目负责人 Milan Kovac 辞职，Ashok Elluswamy 接任 |
| **2026.02** | 转向月球基地计划 |

### 1.3 技术规格

| 参数 | 数值 |
|------|------|
| **身高** | 5 ft 8 in (173 cm) |
| **体重** | 125 lb (57 kg) |
| **负载能力** | 45 lb (20 kg) |
| **控制** | 与 Tesla Autopilot 相同的 AI 系统 |

**技术**：
- 特点线性执行器使用行星滚柱螺杆技术（替代传统滚珠丝杠）
- 更高的冲击负载能力

### 1.4 迭代版本

| 版本 | 发布时间 | 自由度 (手部) | 特点 |
|------|----------|----------------|------|
| **Gen 1** | 2022 | 未知 | 原型 |
| **Gen 2** | 2023.09 | 11 DOF | 更纤细、改善的手部、瑜伽姿势 |
| **Gen 3** | 2024 | 22 DOF | 进一步改进 |

### 1.5 商业化时间线

| 时间 | 计划 |
|------|------|
| **2025** | 有限生产（1,000+ 台在 Tesla 设施内使用） |
| **2026** | 对外销售给其他公司 |
| **2030** | 载人登月任务 |

### 1.6 争议与批评

| 来源 | 观点 |
|------|------|
| **Bloomberg** | "mission creep"，偏离清洁能源主业 |
| **Washington Post** | "夸大时间表，过度承诺" |
| **The Verge** | "充满幻想的想法，能否成真未知" |
| **Gary Marcus** | "2023年底前不会有机器人能做所有人类任务" |
| **Deutsche Welle** | 专家称"完全彻底的骗局" |
| **Rodney Brooks** | "通用人形助手是纯幻想" |

### 1.7 核心争议点

1. **远程操作质疑**
   - 宣传视频中的任务被指需要远程操作（teleoperation）
   - 竞争对手发布对比视频展示自主能力

2. **技术领先性存疑**
   - 评论家指出：其他机器人项目早已实现类似功能
   - Tesla 能否"超越"其他公司存疑

3. **时间表可靠性**
   - Musk 以往承诺多次未能兑现
   - 2023年量产目标未能实现

## 四、争议与安全问题

### 4.1 Unitree 安全争议

| 时间 | 事件 | 详情 |
|------|------|------|
| **2022.08** | 俄罗斯军队使用指控 | Unitree 否认 |
| **2023.09** | 美国海军测试 | 装载 M72 LAW 火箭筒 |
| **2024.05** | 中国军队使用指控 | 柬埔寨军演，自动步枪 |
| **2024.04** | 后门指控 | 被指有远程访问后门 |
| **2025.09** | 安全漏洞 | G1 收集数据不通知用户 |
| **2025.09** | BLE 漏洞 | 近距离可完全控制 |

**安全研究结论**：
- G1 收集多模态传感器数据，可能用于网络攻击
- BLE 漏洞可感染附近其他机器人
- Unitree 拒绝评论

### 4.2 地缘政治因素

**美国审查**：
- 2025年5月：众议院中国委员会要求 FCC、商务部、国防部调查
- 理由：涉嫌与中国军方关联（军民融合）

**公司回应**：
- 不向中国人民解放军销售产品
- 后门是漏洞，已修复

### 4.3 军事应用

| 应用 | 地点 | 说明 |
|------|------|------|
| **乌克兰战场** | 乌克兰 | 探索建筑、战壕（英国公司采购改装） |
| **柬埔寨军演** | 柬埔芽 | 装载自动步枪 |

## 六、未来展望

### 6.1 技术趋势

| 趋势 | 说明 |
|------|------|
| **成本下降** | 从 $30,000 → $16,000 趋势 |
| **自主能力提升** | 从远程操作到自主任务 |

### 6.2 商业化路径

```
消费级（$10K-30K）
    ↓
工业级（$50K-100K）
    ↓
军事/特种（$100K+）
```

### 6.3 挑战

| **监管** | 地缘政治审查加强 |

## 九、总结

### 核心洞察

1. **Tesla Optimus**：
   - 雄心勃勃但争议不断
   - 技术自主性受质疑（远程操作指控）
   - 量产时间表多次推迟

2. **中国机器人（Unitree）**：
   - 成本优势明显（$16,000 vs $30,000）
   - 商业化领先（已量产）
   - 安全隐患和地缘政治风险

3. **行业状态**：
   - 从研究走向商业化
   - 成本快速下降
   - 安全和伦理问题凸显

### 投资/关注建议

| 方向 | 建议 |
|------|------|
| **消费级** | Unitree G1 ($16,000) 是当前最佳选择 |
| **工业级** | 等待 Tesla Optimus 2025 量产 |
| **研究级** | Unitree H1、Boston Dynamics（历史） |
| **警惕** | 远程操作能力伪装为自主能力 |
| **安全** | 关注 BLE/后门漏洞 |
