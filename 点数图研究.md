# 点数图 (Point and Figure Chart) 研究

> 日期：2026-02-11
> 来源：Wikipedia, Technical Analysis History

---

## 核心概述

**点数图 (P&F)** 是一种**不基于时间**的技术分析图表方法。

- **X 列**：代表价格上涨
- **O 列**：代表价格下跌
- **横轴**：不表示时间，只表示价格变化的列数

## 核心概念

### 1. 框 (Box Size)

每个 X 或 O 代表的价格变动幅度：
- **低价股**：$0.50 - $1.00
- **中价股**：$1.00 - $2.00
- **高价股**：$2.00 - $5.00

**作用**：过滤微小波动，减少噪音

### 2. 转向 (Reversal)

价格向相反方向变动多少个框才新开一列：

| 类型 | 说明 | 特点 |
|------|------|------|
| **1-box 转向** | 反向变动1个框 | 敏感，信号多 |
| **3-box 转向** | 反向变动3个框 | 稳定，常用 |

### 3. 45度趋势线

在方格纸上绘制 45 度线来定义趋势：
- **上升趋势线**：从重要低点向右上方延伸
- **下降趋势线**：从重要高点向右下方延伸

---

## 相比传统K线的优势

| K线图 | 点数图 |
|-------|--------|
| 基于时间 | 基于价格变动 |
| 信号频繁但噪音多 | 信号过滤，趋势清晰 |
| 易受短期波动干扰 | 忽略微小波动 |
| 显示开盘/收盘/最高/最低 | 只显示方向和幅度 |

---

## 价格目标测算

### 1. 垂直计数法 (Vertical Count)

测量从高点/低点出发的推动幅度，直接投影得出目标：

```
目标 = 低点 + (推动幅度 × 转向倍数)
```

### 2. 水平计数法 (Horizontal Count)

测量横盘整理区的宽度，用于测算突破后的目标：

```
目标 = 整理区突破点 + 整理区宽度
```

## 常见形态

| 形态 | 信号 | 交易含义 |
|------|------|----------|
| **双底 (Double Bottom)** | O列触及前低后反弹 | 买入信号 |
| **双顶 (Double Top)** | X列触及前高后回落 | 卖出信号 |
| **突破 (Breakout)** | X列突破阻力位 | 上涨信号 |
| **跌破 (Breakdown)** | O列跌破支撑位 | 下跌信号 |
| **三高/三低 (Triple Top/Bottom)** | 三次测试阻力/支撑 | 强信号确认 |

## 现代应用

### 百分比框 (Percentage Box)

不用固定金额，而用百分比设置框大小：
- **优点**：无论股价高低，敏感性保持一致
- **适用**：长期分析和跨市场比较

### 自动化

- 1980s：Jeremy du Plessis 用电脑实现点数图自动化
- 现在多数交易软件都支持 P&F 图表

---

## 优点总结

1. **过滤噪音**：不显示小幅波动
2. **趋势清晰**：45度线直观显示趋势
3. **目标明确**：垂直/水平计数法提供目标位
4. **客观性强**：规则明确，减少主观判断
5. **时间独立**：不显示时间轴，适合长周期分析

---

## 局限性

1. **入场时机较晚**：过滤噪音的同时可能错过最佳入场点
2. **需要手工计算**：传统方法需要每天更新
3. **参数选择主观**：框大小和转向倍数需要经验
4. **不显示成交量**：缺少量能信息

---

## 总结

点数图核心：**去芜存菁，聚焦趋势**

- 用价格变动代替时间轴
- 用 X/O 符号代替 K 线
- 适合识别中期趋势和测算目标
- 与其他技术分析工具配合使用效果更佳
