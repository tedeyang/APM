## 四、避坑指南

### 4.1 智商税产品

| 产品 | 智商税原因 | 替代方案 |
|------|-----------|----------|
| **自动炒菜机** | 翻炒不均匀、清洁麻烦 | 电压力锅焖饭 |
| **早餐机** | 功能单一、使用频率低 | 多士炉+电磁炉 |
| **酸奶机** | 成功率低、性价比差 | 破壁机带酸奶功能 |
| **榨油机** | 操作复杂、出油率低 | 买现成的 |
| **面条机** | 清洁麻烦、使用频率低 | 买鲜面条 |

### 4.2 选购误区

| 误区 | 正确做法 |
|------|----------|
| **贵就是好** | 选对功能，不选贵功能 |
| **功能越多越好** | 选核心功能常用的 |
| **进口一定好** | 国产性价比更高（除特定品类） |
| **容量越大越好** | 按家庭人数选（3人5L、4人6L） |
| **促销必买** | 先做功课，促销只是便宜一点 |

### 4.3 省钱技巧

1. **套装更便宜**：买锅具套装、厨电套装
2. **等大促**：618、双11力度大
3. **买上一代**：新款上市，老款打折
4. **选国产品牌**：九阳、美的、苏泊尔性价比高
5. **一机多用**：选多功能款，少买单一功能
